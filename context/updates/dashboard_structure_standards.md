# QBank Dashboard Structuring Standards

## Category
New information learned via web research.

## Description
When designing dashboards for Indian medical postgraduate entrance exams (NEET PG, INI-CET, FMGE), the UI must match the actual curriculum layout of MBBS students to feel authentic and avoid the look of generic AI templates. 

## Key Structuring Standards
1. **Professional Phases Grouping**:
   Instead of a simple alphabetical list, group the 19 standard MBBS subjects into the four distinct professional phases:
   - **Pre-Clinical (1st Professional Year)**: Anatomy, Biochemistry, Physiology
   - **Para-Clinical (2nd & 3rd Professional Year)**: Pharmacology, Pathology, Microbiology, Forensic Medicine, Social & Preventive Medicine (SPM)
   - **Clinical Specialties (Allied Sciences)**: Ophthalmology, ENT, Pediatrics, Orthopedics, Dermatology, Psychiatry, Radiology, Anesthesia
   - **Core Clinicals (Final Professional Year)**: General Medicine, General Surgery, Obstetrics & Gynecology
2. **Active Recall Entry Points**:
   Aspirants spend the majority of their revision loops in bookmarks and incorrect questions. Dashboards should present direct action links ("Resolve Incorrects", "Review Bookmarks") to start quick-practice sets for these filters.
3. **Daily Target Loop**:
   Include a daily progress ring/bar counting the number of solved MCQs against a customizable target (typically 50-100 questions/day).
4. **Target Exam Context**:
   Allow switching the active target exam (NEET PG, INI-CET, FMGE) to adjust exam-specific weightings and visual focus, preserving user state locally.
